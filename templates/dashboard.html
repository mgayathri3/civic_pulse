Help Center

Getting Started

Welcome to the CivicPulse Admin Dashboard. This help guide will assist you in navigating and using the dashboard effectively.

Managing Complaints

To view and manage complaints:

  • Click on the "Complaints" tab in the navigation menu
  • Use the search bar to find specific complaints
  • Click the eye icon to view complaint details
  • Update the status of complaints using the status dropdown
  • Upload images related to the complaint for documentation

Filtering Data

Use the filter button to filter complaints by:

  • Status (New, Assigned, In Progress, Resolved)
  • Priority (High, Medium, Low)
  • Category (Roads, Water, Electricity, etc.)
